# My Full-Stack Project

## Description
This is a full-stack web application built with Laravel, React, and SQL.  
The project demonstrates a dynamic web application with a PHP backend (Laravel) and a React frontend connected to a database.

---

## Technologies Used
- Backend: Laravel (PHP)
- Frontend: React.js
- Database: MySQL (or compatible SQL database)
- Others: Composer, Node.js, npm

---

## Installation & Setup

Follow these steps to run the project locally:

### 1️⃣ Database
- Import the provided SQL file into your MySQL database (using phpMyAdmin or MySQL Workbench).  

### 2️⃣ Laravel Backend
1. Open terminal inside the laravel project folder.
2. Start the laravel server:
   php artisan serve

### 3️⃣ React Frontend
1. Open terminal inside the React project folder.
2. Start the React development server:
   npm start


## Features
 1. Full-stack architecture with Laravel + React
 2. Database-driven dynamic content
 3. RESTful API for frontend-backend communication
 4. Responsive and interactive UI
